# Семейный бюджет

Веб-приложение для учёта семейных доходов и расходов в браузере. Данные хранятся локально на устройстве пользователя (`localStorage`), без сервера и без отправки финансовой информации в сеть.

## Возможности

- Добавление доходов и расходов с описанием и опциональной категорией
- Автоматический пересчёт общего дохода, расхода и баланса
- Таблицы по доходам и расходам, удаление отдельных операций и массовая очистка
- Светлая и тёмная тема (настройка сохраняется в браузере)
- Приветственная страница с описанием проекта и переходом к калькулятору

## Структура проекта

| Файл / папка | Назначение |
|--------------|------------|
| `index.html` | Главная страница (лендинг) |
| `budget.html` | Калькулятор бюджета |
| `css/theme.css` | Общие CSS-переменные темы и кнопка переключения темы |
| `css/landing.css` | Стили только для лендинга |
| `README.md` | Документация репозитория |

## Как открыть сайт

1. **Просто файлы:** откройте в браузере файл `index.html` (двойной клик или «Открыть с помощью…»). Для корректных путей к CSS откройте именно главную страницу из папки проекта.

2. **Локальный сервер (рекомендуется при разработке):**

   ```bash
   cd "путь/к/Калькулятор"
   python3 -m http.server 8080
   ```

   Затем в браузере: `http://localhost:8080/`

## Требования

- Современный браузер с поддержкой ES5+, CSS custom properties и `localStorage`.
